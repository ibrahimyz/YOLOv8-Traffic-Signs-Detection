# YOLOv8 Real-Time Traffic Signs Detection
This project performs real-time traffic sign detection using models trained on the Türkiye Traffic Sign Dataset (TTVS) with the YOLOv8 framework.

There are a total of 11,453 distinct traffic sing images, each with its corresponding label file. All images and labels are split into 8,017 for training, 1,719 for testing, and 1,717 for validation. YOLOv8n and YOLOv8s models were used during training.
The models’ precision, recall, F1-score, and mAP50 values were compared to evaluate their performance.


# YOLOv8 ile Trafik İşaretlerinin Gerçek Zamanlı Tespiti
Bu projede Türkiye Trafik Veri Seti (TTVS) ve YOLOv8 kullanılarak eğitilen modeller ile gerçek zamanlı trafik işareti tespiti yapılmıştır. 

Birbirinden farklı, toplam 11.453 levha görüntüsü ve her birinin etiket dosyası bulunmaktadır. Tüm görüntüler ve etiketler; 8.017 train, 1.719 test, 1.717 validation olacak şekilde ayrılmıştır. Eğitim sırasında YOLOv8n ve YOLOv8s modelleri kullanılmıştır.
Modellerin kesinlik (precision), duyarlılık (recall), F1-skor ve mAP50 değerleri; performanslarını değerlendirmek amacıyla karşılaştırılmıştır.
